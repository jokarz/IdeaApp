# Idea CRUD

CRUD demostration

### Endpoints

Restful end points:
```
Create - /api/idea (HttpPost)
Read - /api/idea (HttpGet), /api/idea/{id} (HttpGet)
Update - /api/idea/{id} (HttpPut)
Delete - /api/idea/{id} (HttpDelete)
```

Frontend path for each corresponding CRUD:
```
Create - /
Read - /, /idea, /idea/{id}
Update - /idea/{id}
Delete - /idea/{id}
```
